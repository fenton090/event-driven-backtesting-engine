# Event-Driven Backtesting Engine (C++)

## Overview
Built a modular event-driven trading system simulating real-world trading workflows.

## Features
- Event-driven architecture (Market, Signal, Order, Fill)
- Strategy plug-in system
- Portfolio and risk management
- Execution simulation
- Performance metrics:
  - PnL
  - Sharpe Ratio
  - Max Drawdown

## Architecture
Market → Strategy → Portfolio → Execution → Portfolio → Analytics
